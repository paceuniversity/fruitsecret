fruitsecret
===========
Fruit Secret, comme son nom l'indique c'est une application mobile permettant de mieux connaitre un fruit. A savoir, son origine, type de vitamines contenant dans le fruit, à quel type de maladie que ce fruit peut soigner ou sa carence peut provoquer et à quelle heure faudrait-il le consommer pour éviter des retombées négatives. Parce que certains fruits ne sont pas recommandés la nuit, car ils peuvent provoquer un manque de sommeil. Cette application sera développée en jquery et du php afin d’interagir avec une base de données. Cette base permettra d'avoir beaucoup plus d.informations sur le fruit et stockera les suggestions ou les commentaires des différents utilisateurs de l'application.

Cette application est dédié à tous catégories de personne, enfants ou adulte, femme ou enfant, jeune ou personne en age car la santé ou le savoir qui découle de ces fruits n'ont pas de type d'age. Mais concerne plus les utilisateurs francophones pour le moment car elle est écrite en français.

Pour la réalisation de cette dernière, les sources de données que nous avons prévenu sont les suivantes:

**le web:**
* Wikipedia
* google
